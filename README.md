Під час прокту ми використовували такі бібліотеки:  
yfinance - для отримання історичних даних про акції Apple.   
pandas - для роботи з DataFrame, очищення даних та розрахунків.
matplotlib - для побудови графіку результатів.

Спочатку ми завантажили історичні дані за 2025 рік, 
потім дані були перевірені, очищені від пропущених значень 
та організовані у DataFrame.  

Далі були розраховані дві прості ковзні середні
(Simple Moving Average):  

короткострокова SMA(на 20 днів)  
довгострокова SMA(на 50 днів)  

Торгові сигнали формуються на основі їх перетину:  

Hold - нових сигналів не було  
Buy - SMA_short > SMA_long  
Sell - SMA_long > SMA_short  

Після цього була розрахована дохідність стратегії, 
кінцева вартість інвестиції та побудований графік 
із ціною акцій, SMA та точками купівлі та продажу.

**○ Короткі висновки щодо ефективності
вашої торгової стратегії.**  

За обраний період зі стартовим капіталом у \$10000 нам 
вдалося отримати \$2580.82. Обрана стратегія ефективно 
обробляє сигнали та дійсно дозволяє заробляти. 
Однак іноді стратегія зазнавала збитків через несвоєчасну 
покупку/продажу, що є однією з проблем методу Moving Average.
Тим не менш оскільки акції Apple мають загальний тренд на 
зріст, тому стратегія спрацювала і капітал збільшився
